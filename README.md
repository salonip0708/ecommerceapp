# Flutter Product List App

A simple Flutter app that fetches product data from Fake Store API and displays products with image, title, price, and favorite icon in a clean UI.

## Features
- Fetch product data from API
- Display product image, title, and price
- Favorite icon for each product
- Clean and responsive UI
- Uses HTTP package

## Technologies Used
- Flutter
- Dart
- HTTP API
- Fake Store API

## How to Run
1. Install Flutter SDK
2. Clone this repository
3. Run `flutter pub get`
4. Run `flutter run`

## Dependencies
- http
